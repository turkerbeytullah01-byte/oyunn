## ⚡ HIZLI BAŞLANGIÇ (5 Dakika)

### 1. Dosyaları İndirin
Bu klasördeki tüm dosyaları bilgisayarınıza indirin.

### 2. Unity Kurun
- [unity.com/download](https://unity.com/download) adresinden Unity Hub'ı indirin
- Unity 2022.3 LTS veya daha yenisini kurun
- **Android Build Support** modülünü kurmayı unutmayın!

### 3. Projeyi Açın
- Unity Hub → Open → Bu klasörü seçin
- Veya: Yeni 3D proje oluşturun ve dosyaları içine kopyalayın

### 4. Build Alın
```
File → Build Settings → Android → Build
```

### 5. Telefona Kurun
- APK dosyasını telefonunuza kopyalayın
- Kurun ve oynayın!

## ❓ SIK SORULAN SORULAR

**S: Unity bilmiyorum, yapabilir miyim?**
C: Evet! Temel bilgiler 1-2 günde öğrenilir. YouTube'da "Unity beginner tutorial" arayın.

**S: Build almak ücretli mi?**
C: Hayır! Unity Personal (ücretsiz) sürümü yeterli.

**S: Google Play'de yayınlamak ücretli mi?**
C: Evet, Google Play Developer hesabı için tek seferlik $25 ücret var.

**S: iPhone için de yapabilir miyim?**
C: Bu proje sadece Android için yapılandırıldı. iOS için ek ayarlar gerekir.

**S: Kodları değiştirebilir miyim?**
C: Evet! Tüm kodlar açık ve değiştirilebilir.
